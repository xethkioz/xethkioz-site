# XETHKIOZ V7 Clean Restart Alpha 2

Base: Web_GITHUB(25).zip.

Objetivo: pulir la web existente sin perder el corazón original de XETHKIOZ, reduciendo paneles, simplificando navegación y dejando tres portales principales + Green Node oculto.

## Cambios aplicados

- Home conservada sobre la estética XETHKIOZ violeta/naranja.
- Eliminado el enfoque de dashboard visible en portada.
- Header convertido en panel mínimo superior derecho: idioma, audio visual y cuenta.
- Eliminado el menú superior clásico en la experiencia principal.
- Green Wisp convertido en acceso visual oculto dentro de la escena principal.
- Eliminado el Wisp flotante con textos/hints para no explicar el secreto al usuario.
- Portales principales mantenidos:
  - Juegos
  - Ciencia y Tecnología
  - Memes, Ocio y Diversión
- Green Node aislado como experiencia verde independiente.
- Idiomas reducidos a ES/EN. Se elimina chino de la base activa para esta etapa.
- Footer y elementos comunitarios flotantes quedan fuera del Home y de los portales aislados.
- Build validado.
