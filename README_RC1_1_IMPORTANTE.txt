XETHKIOZ v4.0 RC1.1 — Network + Science Lab + Green Node

Base: v4.0.0-alpha.4 / RC1
Build probado en entorno Linux con:
npm install
npm run build

No incluye:
- .env
- .git
- node_modules
- dist

Antes de reemplazar:
1. Guardar tu .env actual.
2. Reemplazar carpeta Web_GITHUB.
3. Copiar .env anterior.
4. Ejecutar:
   npm install
   npm run build
   npm run dev

Rutas nuevas/importantes:
- /
- /science
- /green-node
- /news-engine
- /roles
- /cms

Green Node:
- Se accede por wisp verde flotante.
- También se puede abrir directamente con /green-node.
- Incluye video banner en public/videos/green-node-banner.mp4.
