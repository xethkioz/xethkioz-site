# XETHKIOZ v4.0 RC1.1 — Network + Science Lab + Green Node

## Objetivo

Convertir la arquitectura actual en **XETHKIOZ Network**, un ecosistema modular con portales especializados que comparten perfiles, roles, reputación, CMS y Supabase.

## Cambios principales

- Se agrega `XETHKIOZ_NETWORK_PORTALS` como mapa oficial del ecosistema.
- Se transforma `/science` en una división más formal y profesional.
- Se agrega `/green-node` como sección oculta/experimental.
- Se agrega un wisp verde flotante que funciona como Easter Egg de acceso a Green Node.
- Se integra `green-node-banner.mp4` como hero visual de la sección hack/cyberpunk.
- Se agrega migración SQL para:
  - `network_portals`
  - `science_reports`
  - `green_node_entries`
  - `green_node_access_logs`

## Reglas editoriales

### Science Lab

Debe trabajar con:
- fuentes;
- nivel de evidencia;
- informes más largos;
- tono profesional;
- menor estética gamer;
- enfoque educativo y verificable.

### Green Node

Debe trabajar con:
- programación;
- Linux/Ubuntu;
- open source;
- ciberseguridad defensiva;
- OSINT responsable;
- misterios y teorías como análisis documental.

No debe promover hacking ofensivo ni presentar hipótesis como hechos.
