# XETHKIOZ V7 Clean Restart Alpha 5 — Visual Portal Base

## Objetivo
Fijar una base visual cercana al concepto aprobado: portales oscuros estilo MMORPG, Green Wisp como Easter Egg y núcleo XETHKIOZ con dragón/energía.

## Cambios principales
- Reemplazo del Home anterior por una escena más inmersiva.
- Portales con vórtice animado, runas y marco oscuro.
- Wisp flotante sin explicación visible.
- Dragón simbólico con ojos pulsantes y aura violeta.
- Animaciones CSS livianas para no cargar el sistema.

## No incluido
- No se agregó CMS.
- No se agregó SQL.
- No se cambió Supabase.
- No se agregaron secciones nuevas.

## Criterio
Primero se define la experiencia visual base. Luego se agregan funciones sobre esta base.
