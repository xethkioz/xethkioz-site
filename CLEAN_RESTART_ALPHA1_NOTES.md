# XETHKIOZ V7 Clean Restart Alpha 1

Esta entrega toma la web anterior como base y la simplifica.

No es una web nueva desde cero: es una recuperación del corazón original con menos paneles y mejor foco.

## Rutas principales
- `/` núcleo del ecosistema.
- `/gaming` portal Juegos.
- `/science` portal Ciencia y Tecnología.
- `/fun` portal Memes/Ocio/Diversión.
- `/green-node` zona oculta Green Node.

## Regla de UX
Cada portal funciona como mundo independiente con botón para volver al núcleo.
