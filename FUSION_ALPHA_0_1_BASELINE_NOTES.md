# XETHKIOZ Fusion Alpha 0.1 — Baseline

## Objetivo

Congelar una base pobre visualmente pero estable para continuar el desarrollo de XETHKIOZ Fusion sin volver a romper producción.

## Decisión

Esta versión parte de Alpha 5, que fue la última base confirmada como compilable y recuperable después del rollback de Fusion Web 1.0.

## Reglas desde esta versión

- No usar imágenes pegadas como interfaz completa.
- No tocar `main` para experimentos visuales.
- No agregar SQL ni Supabase nuevo durante esta etapa.
- No rediseñar la Home sin assets aprobados.
- Mantener tres portales principales y Green Wisp.
- Priorizar funcionamiento real antes que visual premium.

## Próxima fase

Fusion Alpha 0.2 debe enfocarse en limpieza controlada y documentación del mapa actual:

1. Inventario de rutas.
2. Inventario de componentes.
3. Inventario SQL/migraciones.
4. Definición de assets necesarios para Home final.
5. Blueprint visual antes de tocar Home.
